# Coding Problems Collection

This repository contains solutions to various coding problems and algorithms implemented in Python.

## Problems Included

1. **Count Zeros in Range** - Count occurrences of digit 0 in numbers from 1 to N
2. **Group Anagrams** - Group strings that are anagrams of each other
3. **Minimum Moves to Zero** - Find minimum operations to reduce a number to 0
4. **Find Captain's Room** - Identify the unique room number in a hotel scenario
5. **Coin Change Ways** - Count ways to make change for a given amount
6. **Max Balloons** - Maximum times "balloon" can be formed from given characters
7. **Postman Delivery** - Optimize mail delivery route with capacity constraints
8. **Minimize Sum Products** - Rearrange arrays to minimize sum of products
9. **Minimum Chairs** - Calculate minimum chairs needed for event simulation
10. **Longest Increasing Subarray** - Find longest strictly increasing subsequence
11. **Count Negative Numbers** - Count negatives in sorted matrix efficiently
12. **Matrix Addition** - Add two matrices element-wise
13. **Balanced Braces** - Check if braces are properly matched
